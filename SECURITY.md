# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the irab.app website or any
related irab.app service, please report it privately — do **not** open a
public issue.

- **Email:** support@irab.app
- Include a description of the issue, steps to reproduce, and the
  potential impact.

You will receive an acknowledgement within 72 hours. Please allow a
reasonable disclosure window before sharing details publicly.

## Scope

This repository covers the static website served at
[irab.app](https://irab.app). Vulnerabilities in the mobile app or its
backend APIs can be reported to the same address.
